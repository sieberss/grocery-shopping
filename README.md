### Fullstack team project 
(as part of the NeueFische Java Fullstack Bootcamp) 

An application for grocery shopping. You can add items to buy from the database to your shopping list, modify the quantity, and remove items. Quantity 0 is equivalent to removing. The total price is calculated immediately with each change. Login is required to access the shopping list.

## Tech Stack used
* Java
* Spring Boot
* MongoDB
* OAuth2
* JUnit
* React
* Maven 
* Github Actions
* Sonarcloud
